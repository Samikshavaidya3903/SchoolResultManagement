## Overview

School Result Management System is a Java-based console application developed using JDBC and MySQL. It helps manage student academic results efficiently by performing CRUD operations and automatically calculating percentage and grade.

## Features

* Add Result
* View Results
* Search Result By Roll Number
* Search Result By Name
* Update Result
* Delete Result
* Statistics Dashboard
* Automatic Percentage Calculation
* Automatic Grade Calculation
* Input Validation
